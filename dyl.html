114-1 NTU Design Your Life (DYL)

Week 4 Wayfinding Map

  • File

    • Wayfinding Map 是一種回顧與整理個人經驗的視覺化工具，將重要事件依時間脈絡標示在地圖上。透過書寫事件當下的感受、想法與其對現在自己的影響，幫助我們看見成長的軌跡與轉折點。它不只是記錄過去，更是一張引導未來方向的內在導航圖。

    Week 6 天使與魔鬼

  • File

    • 天使與魔鬼是一個幫助我們釐清價值觀的反思練習，透過對比「想成為／靠近」與「不想成為／靠近」的人物來認識自己。天使象徵內心嚮往的理想樣貌與正向特質，魔鬼則提醒我們哪些狀態與行為值得警惕與避免。在兩者的對照之中，我們能更清楚看見自己真正重視的方向與選擇。

    Week 8 夢想版

  • File

    • 夢想版是一種將未來想望具象化的視覺工具，透過圖片與文字，把抽象的夢想轉化為看得見的畫面。它幫助我們整理人生想完成的重要目標，不論是成就、生活體驗或關係，都能被清楚呈現。當夢想被放在眼前，也更容易成為行動的動力與前進的方向。

    Week 13 動動事件

  • File 1
  • File 2

    • 動動事件：兩週行動計劃是一個把想法轉化為實際行動的練習，透過可執行、可觀察的步驟，展開為期兩週的實驗性嘗試。它鼓勵在行動前設定預期，並在過程中記錄實際行為、感受與產生的差距。透過行動與反思的循環，幫助我們從做中學，調整方向並累積真實經驗。

    Week 16 期末發表

  • File

    • 期末發表是一份整合整學期學習與探索成果的總結，透過創作與分享，回顧自己在「設計人生」中的成長軌跡。它邀請我們將能力、信念、行動經驗與重要發現轉化為具體的故事與作品。不只是成果展示，更是一個向自己與重要他人說明「我是誰、我走到哪裡、接下來要往哪裡去」的時刻。
